# 📌 Overview

Welcome to my personal animated portfolio website — a modern, responsive, and interactive web project designed to showcase my skills, projects, achievements, and journey as a web developer.

This portfolio focuses on delivering a clean UI/UX experience with smooth animations, responsive layouts, and visually engaging sections. The website is built using core frontend technologies without relying on heavy frameworks, making it lightweight and fast.

Whether viewed on desktop, tablet, or mobile devices, the portfolio adapts seamlessly to provide a smooth browsing experience.

---

# ✨ Features

## 🌙 Dark Mode Toggle

Switch between light and dark themes for a better viewing experience.

## 🎯 Smooth Animations

Modern transitions and animations improve the visual appeal and user interaction.

## 📱 Fully Responsive Design

Optimized for all screen sizes including mobile, tablet, and desktop.

## 🧭 Easy Navigation

Simple and user-friendly navigation for smooth scrolling across sections.

## 💼 Project Showcase

Dedicated section to highlight projects with descriptions and live/demo links.

## 📞 Contact Section

Allows visitors and recruiters to connect easily.

## ⚡ Lightweight & Fast

Built using pure HTML, CSS, and JavaScript for better performance.

## 🎨 Modern UI Design

Clean layout with attractive colors, typography, and visual hierarchy.
